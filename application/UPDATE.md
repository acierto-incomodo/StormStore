# 🚀 StormStore v2.1.10 - Interfaz y Ajustes

## 📝 Registro de Cambios

Esta actualización introduce mejoras críticas en la visualización de la interfaz, nuevos controles de usuario para el inicio del sistema y una lógica de actualización más inteligente para asegurar que nunca te pierdas las novedades importantes. También agrega el apartado de soporte, apartado de novedades y la eliminación del ajuste de actualizaciones automáticas.
