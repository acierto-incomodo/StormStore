# 🚀 StormStore v2.0.2

## 📝 Registro de Cambios

### ✨ Novedades

- **Nueva Barra de Título:** Se ha implementado el _Window Controls Overlay_ de Electron, integrando los controles nativos de Windows directamente en el diseño de la app.
- **Áreas de arrastre mejoradas:** Se han definido regiones de arrastre personalizadas para una navegación más fluida.
- **Soporte para Epic Games:** Se ha habilitado la fuente de Epic Games para explorar juegos de tu biblioteca (Función en fase beta 🧪).
- **Motor de descargas optimizado:** Ahora compatible con archivos `.zip` y volúmenes divididos (`.zip.001`, `.zip.002`, etc.) mediante la integración de 7-Zip.
- **Extras:** Instalación masiva de más de 30 dependencias de NodeJS para mayor velocidad.
- **Compatibilidad:** Mejoras específicas para sistemas Windows 10 y 11.
- **Librerías:** Incorporación de nuevos archivos `.dll` esenciales para la ejecución de procesos internos.

### 🛠️ Correcciones de Errores

- **Estabilidad:** Se han corregido múltiples bugs que afectaban la funcionalidad principal.
- **Actualizador:** Se ha reparado el sistema de actualizaciones automáticas para garantizar que siempre estés en la última versión.
- **Optimización:** Refactorización de código en el proceso principal para reducir el consumo de recursos.

### 🗑️ Cambios y Eliminaciones

- **Linux (Wine):** Se ha retirado la compatibilidad oficial mediante Wine en Linux para priorizar una experiencia nativa y estable en Windows.

### 🆙 Actualizaciones Técnicas

- **Dependencias:** Todos los componentes de NodeJS han sido elevados a sus versiones más recientes.
- **Herramientas externas:** Se ha actualizado el motor de extracción `7zr.exe` a la última versión disponible.
