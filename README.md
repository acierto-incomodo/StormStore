![StormStore Photo](https://github.com/acierto-incomodo/StormStore/blob/main/stormstore.png)

  

## Install repo on the sistem


    curl -fsSL https://raw.githubusercontent.com/acierto-incomodo/StormStore/main/install.sh | sudo bash


## Install repo on the sistem and all packages


    curl -fsSL https://raw.githubusercontent.com/acierto-incomodo/StormStore/main/install-all.sh | sudo bash


## Remove repo on the sistem


    curl -fsSL https://raw.githubusercontent.com/acierto-incomodo/StormStore/main/remove.sh | sudo bash

Only remove de repository, not the packages.



## Program list:

CardinalAI MultiModel APP (cardinal-ai-dualmodel-app)

    sudo apt install cardinal-ai-dualmodel-app

  

  

WhatsApp Web (whatsapp-web)

    sudo apt install whatsapp-web

  

  
  

PairDrop APP (pairdrop)

    sudo apt install pairdrop

  

  
  

MyJonCraft SGS Config Transfer (data-exporter)

    sudo apt install data-exporter

  

  
  

MultiAI (multiai)

    sudo apt install multiai
  
  

TheShooter (theshooterlauncher)

    sudo apt install theshooterlauncher
